# MacVendor.io Privacy Policy

Last updated: 2024-11-28

## What We Collect
- Lookup requests: MAC address, timestamp, client IP, user agent, API key identifier.
- Usage statistics: request counts, endpoints accessed, response times (aggregated).

## How We Use Data
- Deliver MAC vendor lookup responses.
- Prevent abuse through rate limiting and ensure service availability.
- Improve service quality and accuracy.

## Retention
- Lookup logs (IP, timestamp, MAC) are retained for up to 30 days for rate limiting and abuse prevention, then deleted.
- Aggregated usage statistics are kept for service monitoring.

## Data Minimization
- Only essential data is collected and stored.
- Request bodies are not logged.

## Sharing
- We do not sell or share personal data with third parties.

## Security
- API keys are stored as salted hashes.
- TLS encryption for all communications.
- Data is encrypted at rest.

## Your Rights
- Request deletion of your API key and associated data.
- Contact: privacy@macvendor.io

## Changes
- Policy updates will be announced in release notes.
